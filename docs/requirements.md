# Fingerprint Launcher 需求文档

## 1. 项目概述

Fingerprint Launcher 是一个指纹浏览器启动器桌面应用，用户可以管理多个浏览器启动项，每个启动项关联账号、指纹ID、用户文件夹和浏览器启动参数。

## 2. 功能需求

### 2.1 启动项管理

- **创建启动项**: 用户可以新建启动项，填写以下信息：
  - 启动项名称
  - 指纹ID (fingerprint_id)
  - 用户数据文件夹路径 (user_data_dir)
  - 关联的账号列表
  - 每个浏览器的启动参数配置
- **编辑启动项**: 修改已有启动项的所有字段
- **删除启动项**: 删除指定启动项
- **列表展示**: 分页展示所有启动项，支持搜索
- **启动浏览器**: 点击启动项中的浏览器图标，使用对应参数启动浏览器

### 2.2 浏览器管理 (系统设置)

- **添加浏览器**: 配置浏览器名称、可执行文件路径、图标
- **编辑浏览器**: 修改已有浏览器配置
- **删除浏览器**: 移除浏览器配置
- 浏览器添加后，所有启动项中自动出现该浏览器的图标和名称
- 每个启动项中的浏览器参数必须有设置值，不允许为空/默认

### 2.3 账号管理

- **平台管理**: 添加/编辑/删除平台 (如 GitHub、OpenAI 等)
- **账号管理**: 在平台下添加/编辑/删除账号，包含：
  - 用户名
  - 密码 (加密存储)
  - 备注
- **展示方式**: 支持树形结构 (平台 -> 账号) 和列表结构两种视图切换
- 启动项可关联多个账号

### 2.4 系统设置

- 浏览器配置管理
- 数据存储路径配置
- 通用设置

## 3. 非功能需求

- 数据持久化使用 SQLite
- 密码字段加密存储
- 支持 Windows / macOS / Linux
- 界面响应流畅
